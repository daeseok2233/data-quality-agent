# 데이터 품질 점검 에이전트

## 📌 프로젝트 개요
매일 새로 들어오는 CSV 데이터의 품질(결측치, 스키마, 이상치)을 자동으로 점검하고,  
AI 에이전트가 사람이 읽을 수 있는 리포트를 생성합니다.

## 🎯 프로젝트 목표
데이터 품질 검증 및 리포트 생성을 자동화하여 사람이 직접 검수하지 않아도 되는 시스템을 구축한다.

## ⚙️ 주요 기능
1. CSV 데이터 품질 검증 (결측치, 이상치, 스키마 비교)  
2. 품질 점검 결과 요약 및 저장 (JSON / Markdown)  
3. AI 에이전트를 통한 리포트 자동 생성  
4. GitHub Actions 자동 실행 및 리포트 업로드  
5. pytest 기반 품질 테스트  

## 🧩 기술 스택
- Python (pandas, openai, langchain)
- GitHub Actions (자동화)
- pytest (테스트)
